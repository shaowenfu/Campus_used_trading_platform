# 校园二手交易平台管理端

## 项目介绍
本项目是校园二手交易平台的管理端网页客户端，基于Vue框架开发。主要用于管理员对平台的运营管理，包括商家管理、商品管理、订单管理、分类管理等功能。

## 技术栈
- Vue.js
- Vue Router
- Vuex 
- Element UI
- Axios

## 功能模块
### 1. 管理员模块
- 管理员登录/退出
- 修改密码

### 2. 商家管理模块
- 商家列表(分页查询)
- 新增商家
- 编辑商家信息
- 启用/禁用商家账号
- 查看商家详情

### 3. 商品管理模块
- 商品列表(分页查询)
- 根据分类查询商品
- 商品上架/下架
- 编辑商品信息
- 查看商品详情

### 4. 订单管理模块
- 订单列表(条件搜索)
- 查看订单详情
- 取消订单
- 订单统计

### 5. 分类管理模块
- 分类列表(分页查询)
- 新增分类
- 编辑分类
- 删除分类
- 启用/禁用分类

### 6. 新闻管理模块
- 新闻列表
- 新增新闻
- 编辑新闻
- 删除新闻
- 启用/禁用新闻
