# 校园二手交易平台小程序

## 项目结构

```
├── pages/
│   ├── index/                 // 首页
│   ├── category/             // 分类页面
│   ├── cart/                 // 购物车
│   ├── user/                 // 个人中心
│   ├── address/             // 地址管理
│   │   ├── list            // 地址列表
│   │   ├── edit            // 地址编辑
│   │   └── add             // 新增地址
│   ├── goods/              // 商品相关
│   │   ├── list           // 商品列表
│   │   └── detail         // 商品详情
│   └── order/              // 订单相关
├── components/             // 公共组件
├── utils/                  // 工具函数
├── api/                    // API接口
├── styles/                 // 公共样式
└── app.json               // 小程序配置文件
```
